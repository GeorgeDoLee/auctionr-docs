# Technology Stack

- **Framework**: [ASP.NET Core 8](https://learn.microsoft.com/en-us/aspnet/core/introduction-to-aspnet-core)
- **Architecture**: Onion Architecture, Event-Driven Architecture
- **Patterns**: [CQRS](https://martinfowler.com/bliki/CQRS.html), [Mediator (MediatR)](https://www.nuget.org/packages/MediatR), [Unit of Work](https://learn.microsoft.com/en-us/ef/ef6/fundamentals/unitofwork-pattern)
- **Data Access**: [Entity Framework Core](https://www.nuget.org/packages/Microsoft.EntityFrameworkCore)
- **Real-Time Communication**: [SignalR](https://www.nuget.org/packages/Microsoft.AspNetCore.SignalR)
- **Validation**: [FluentValidation](https://www.nuget.org/packages/FluentValidation)
- **Mapping**: [Mapster](https://www.nuget.org/packages/Mapster)
- **Logging**: [Serilog](https://www.nuget.org/packages/Serilog.AspNetCore)
- **Authentication**: [JWT Bearer Tokens](https://www.nuget.org/packages/Microsoft.AspNetCore.Authentication.JwtBearer)
- **Rate Limiting**: [Microsoft.AspNetCore.RateLimiting](https://www.nuget.org/packages/Microsoft.AspNetCore.RateLimiting)
- **Job Scheduling**: [Hangfire](https://www.nuget.org/packages/Hangfire)
